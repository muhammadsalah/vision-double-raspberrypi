Installation
============

- On the Raspberry Pi, run ```./setup.sh```, and make sure to set "cam_feed=True" in app/main.py @ Line: 26 to enable the Pi Camera
- On a Mac, make sure [homebrew][] is installed, then run ```./setup_macos.sh```

Running
=======

- Run ```./run.sh```

  [homebrew]: http://brew.sh/
